# Dengue-ICU-Pediatric-Analysis

## Objective:
    - Identify clinical factors associated with severity / outcomes
    - using statistical analysis and interpretable ML.

## Key Constraints:
    - Small dataset (~100 samples)
    - High-dimensional clinical variables
    - Risk of overfitting

## Approach:
    - Strong EDA
    - Statistical testing (primary)
    - ML as complementary tool (not main claim)
